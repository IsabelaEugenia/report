# Report+

Sistema de reporte e acompanhamento de ocorrências empresariais desenvolvido em Flutter e Firebase.

## Tecnologias

* Flutter
* Firebase Authentication
* Cloud Firestore
* Google Fonts

## Como executar

1. Instalar Flutter.
2. Executar:

flutter pub get

3. Executar:

flutter run
